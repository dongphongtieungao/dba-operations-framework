## Definition

SQL Server Agent job operations là nhóm tác vụ kiểm tra, điều chỉnh và chuẩn hóa SQL Agent jobs, gồm lịch chạy, lịch sử job, owner, notification và các job bảo trì kiểu Ola Hallengren.

## Variants

- Kiểm tra job đang chạy.
- Chuẩn hóa owner và notification.
- Điều chỉnh lịch hoặc tham số job bảo trì.
