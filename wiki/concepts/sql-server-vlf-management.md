## Definition

SQL Server VLF management là nhóm tác vụ phát hiện, trực quan hóa và giảm số lượng Virtual Log Files trong transaction log để tránh rủi ro vận hành hoặc hiệu năng do log file bị phân mảnh quá mức.

## Variants

- Detect too many VLFs.
- Reduce VLF count.
- Visualizing VLFs.
