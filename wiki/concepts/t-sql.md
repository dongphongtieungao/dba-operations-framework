## Definition

T-SQL, hay Transact-SQL, là dialect SQL của Microsoft dùng để truy vấn, sửa dữ liệu và lập trình trên SQL Server, Azure SQL Database và Azure SQL Managed Instance. T-SQL dựa trên SQL chuẩn nhưng có thêm các phần mở rộng riêng, nên DBA cần phân biệt giữa hành vi chuẩn, hành vi riêng của SQL Server và hành vi phụ thuộc phiên bản.

## Variants

- DML: SELECT, INSERT, UPDATE, DELETE, TRUNCATE và MERGE.
- DDL: CREATE, ALTER, DROP cho định nghĩa object.
- DCL: GRANT, REVOKE và các câu lệnh quyền.
- T-SQL programmability: biến, batch, IF/WHILE, cursor, stored procedure, function, trigger và error handling.
