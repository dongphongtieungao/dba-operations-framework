## Definition

SQL Server backup and restore operations là nhóm tác vụ tạo script phục hồi, theo dõi tiến trình backup/restore, lấy danh sách file backup và cập nhật lịch backup cho database.

## Variants

- Sinh lệnh RESTORE cho một hoặc nhiều database.
- Theo dõi tiến trình BACKUP/RESTORE.
- Điều chỉnh lịch backup.
