## Definition

SQL Server Extended Events monitoring là cách tạo và đọc phiên Extended Events để bắt lỗi, failed logins, blocked processes, truy vấn lớn, cảnh báo execution plan, page splits hoặc hoạt động trên bảng/thủ tục cụ thể.

## Variants

- Monitoring theo sự kiện lỗi hoặc bảo mật.
- Monitoring theo hiệu năng và blocking.
- Tracking có phạm vi hẹp cho một bảng hoặc stored procedure.
