## Definition

Migration assessment là bước phân tích cấu hình, utilization và hiệu năng của nguồn để xác định readiness, blocker, khuyến nghị target, sizing và tác động khi chuyển sang Azure SQL hoặc nền tảng dữ liệu Azure khác.

## Variants

- Azure Migrate assessment: đánh giá ở quy mô estate, có thể dùng dữ liệu discovery.
- Azure Arc readiness assessment: đánh giá readiness liên quan SQL Server enabled by Azure Arc.
