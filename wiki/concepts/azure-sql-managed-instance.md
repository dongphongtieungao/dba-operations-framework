## Definition

Azure SQL Managed Instance là lựa chọn PaaS ở mức instance cho SQL Server trên Azure, thường phù hợp với migration ứng dụng hiện có cần mức tương thích SQL Server cao hơn Azure SQL Database nhưng vẫn muốn giảm gánh nặng quản trị hạ tầng.

## Variants

- General Purpose: phù hợp đa số workload phổ thông.
- Business Critical: hướng đến yêu cầu hiệu năng, độ sẵn sàng và độ trễ cao hơn.
