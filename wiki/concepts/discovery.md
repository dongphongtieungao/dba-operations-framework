## Definition

Discovery là bước thu thập inventory về máy chủ, SQL Server instances, databases, ứng dụng và phụ thuộc liên quan để hiểu phạm vi migration trước khi đánh giá readiness hoặc lập kế hoạch cutover.

## Variants

- Agentless discovery: thu thập bằng appliance hoặc credential mà không cài agent lên server nguồn.
- Data estate discovery: nhìn ở quy mô nhiều server, instance và database.
