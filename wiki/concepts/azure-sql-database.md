## Definition

Azure SQL Database là dịch vụ cơ sở dữ liệu quan hệ được quản lý ở mức database, phù hợp với ứng dụng cloud hiện đại, nhu cầu tự động hóa vận hành và các mô hình như serverless, elastic pools hoặc Hyperscale.

## Variants

- Single database: mô hình quản trị theo từng database.
- Elastic pool: gom nhiều database có nhu cầu tài nguyên biến động.
- Hyperscale: kiến trúc tách compute và storage cho khối lượng dữ liệu lớn hơn.
