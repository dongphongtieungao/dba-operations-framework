## Definition

SQL Server Operations Appendix là phụ lục vận hành chuyển các policy, SOP và runbook chung thành thao tác cụ thể cho SQL Server on-premises hoặc SQL Server on VM. Nó bao phủ cấu hình baseline, backup/restore, Always On, FCI, SQL Agent, DBCC CHECKDB, Query Store, index/statistics maintenance, monitoring, security và evidence.

## Variants

- On-premises SQL Server: DBA kiểm soát sâu máy chủ, storage, network, OS và SQL Server instance.
- SQL Server on VM: vẫn dùng cơ chế SQL Server truyền thống nhưng cần thêm kiểm soát cloud VM, disk, backup và network.
- Always On/FCI estate: nhấn mạnh topology, listener, quorum, failover, replica health và DR drill.
- Maintenance-heavy estate: nhấn mạnh SQL Agent, CHECKDB, index/statistics, backup chain và job evidence.
