## Definition

SQL Server storage and TempDB operations là nhóm script xem kích thước database/table/file, dung lượng trống, shrink file/log, di chuyển data file, phát hiện TempDB lệch kích thước/growth và truy tìm chủ sở hữu temp table.

## Variants

- Phân tích dung lượng database, table và file.
- Điều chỉnh hoặc di chuyển file.
- Kiểm tra TempDB và temp table.
