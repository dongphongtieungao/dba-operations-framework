## Definition

SQL Server on Azure VMs là mô hình chạy SQL Server trên máy ảo Azure, giữ quyền kiểm soát hệ điều hành và cấu hình instance nhiều nhất, đổi lại DBA vẫn chịu trách nhiệm vận hành gần với môi trường IaaS hơn.

## Variants

- SQL Server trên Windows VM.
- SQL Server trên Linux VM.
