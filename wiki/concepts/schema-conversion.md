## Definition

Schema conversion là bước chuyển đổi cấu trúc database và các đối tượng như tables, indexes, views, stored procedures hoặc dữ liệu liên quan khi nguồn và đích khác engine hoặc khác mức tương thích.

## Variants

- Homogeneous migration: ít hoặc không cần chuyển đổi schema.
- Heterogeneous migration: thường cần công cụ hỗ trợ như SQL Server Migration Assistant.
