## Definition

Azure SQL Operations Appendix là phụ lục vận hành chuyển các policy, SOP và runbook DBA chung sang Azure SQL Database và Azure SQL Managed Instance. Nó tập trung vào backup tự động, PITR, Long-Term Retention, failover group, geo-replication, Azure Monitor, diagnostic settings, Query Store, private endpoint, Microsoft Entra ID, TDE, capacity/cost và các lựa chọn thay thế SQL Agent.

## Variants

- Azure SQL Database: database-level PaaS, nhấn mạnh elastic pool, serverless, Hyperscale, Elastic Jobs và external automation.
- Azure SQL Managed Instance: instance-level PaaS, gần SQL Server hơn, có SQL Agent và VNet integration.
- DR-focused Azure SQL: failover group, geo-replication, geo-restore, private endpoint và connection strategy.
- Observability-focused Azure SQL: Azure Monitor, diagnostic settings, Query Store, wait stats, alerts và cost/capacity.
