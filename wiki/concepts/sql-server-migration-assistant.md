## Definition

SQL Server Migration Assistant là nhóm công cụ hỗ trợ đánh giá và chuyển đổi schema/dữ liệu khi migration từ các hệ quản trị khác hoặc từ SQL Server sang các đích trong hệ sinh thái SQL Server/Azure SQL.

## Variants

- Assessment: phát hiện vấn đề tương thích trước migration.
- Conversion: chuyển đổi schema hoặc đối tượng database khi nguồn và đích khác engine.
