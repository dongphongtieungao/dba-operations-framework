## Definition

SQL Server security auditing là nhóm script kiểm tra quyền, login, orphaned users, sysadmin membership, audit file, Windows group membership và kết nối mã hóa/không mã hóa.

## Variants

- Audit quyền và access ở nhiều database.
- Xử lý orphaned users hoặc login không còn hợp lệ.
- Theo dõi audit file, failed login và encryption status.
