## Definition

SQL Server DBA toolbox là tập hợp script, truy vấn và thủ tục hỗ trợ DBA kiểm tra, chẩn đoán, chuẩn hóa và xử lý sự cố trong môi trường SQL Server.

## Variants

- Script đọc trạng thái: truy vấn catalog view, DMV, Query Store, plan cache hoặc Extended Events.
- Script sinh lệnh: tạo câu lệnh RESTORE, ALTER, DROP/CREATE hoặc chuẩn hóa cấu hình để DBA xem trước.
- Script thao tác trực tiếp: thay đổi cấu hình, quyền, job, index, log file hoặc đối tượng database.
