## Definition

Azure SQL là họ sản phẩm cơ sở dữ liệu trên Azure dùng SQL Server database engine, gồm các lựa chọn PaaS và IaaS để cân bằng giữa mức quản trị được Microsoft đảm nhận, mức tương thích SQL Server, quyền kiểm soát hạ tầng và yêu cầu hiện đại hóa ứng dụng.

## Variants

- [[concepts/azure-sql-database]]: dịch vụ ở mức database cho ứng dụng cloud hiện đại.
- [[concepts/azure-sql-managed-instance]]: dịch vụ ở mức instance, hướng đến migration SQL Server với mức tương thích cao.
- [[concepts/sql-server-on-azure-vms]]: SQL Server chạy trên máy ảo Azure, phù hợp khi cần quyền kiểm soát hệ điều hành.
