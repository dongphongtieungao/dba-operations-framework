## Definition

T-SQL programmable objects là các thành phần lập trình trong SQL Server như batch, biến, control-of-flow, cursor, stored procedure, user-defined function, trigger và error handling. Với DBA, chúng quan trọng vì logic nghiệp vụ, job vận hành, trigger audit và code xử lý lỗi thường nằm trực tiếp trong database.

## Variants

- Batch và biến: đơn vị gửi lệnh và giữ trạng thái tạm.
- Control-of-flow: IF/ELSE và WHILE.
- Stored procedure và function: đóng gói logic tái sử dụng.
- Trigger và error handling: phản ứng với thay đổi dữ liệu và xử lý lỗi bằng TRY/CATCH.
