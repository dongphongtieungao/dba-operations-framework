## Definition

SQL Server administration là thực hành vận hành nền tảng SQL Server sao cho database an toàn, sẵn sàng, có thể phục hồi và đạt hiệu năng ổn định. Công việc này trải rộng từ cấu hình instance, quản lý database, giám sát, bảo trì, bảo mật, backup/restore, HA/DR, tự động hóa đến tích hợp cloud.

## Variants

- Quản trị on-premises: DBA chịu trách nhiệm sâu với máy chủ, hệ điều hành, storage, network và SQL Server instance.
- Quản trị hybrid: kết hợp SQL Server tại chỗ với Azure Arc, Azure SQL Managed Instance link hoặc các dịch vụ Azure.
- Quản trị PaaS: tập trung vào database, bảo mật, hiệu năng, chi phí và khả năng phục hồi thay vì quản lý máy chủ.
